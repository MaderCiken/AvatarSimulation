How Does it Work?

Steps

  • Step 1:

    Extracting Face Location

    Locating a face is usually based on the similarity measure results between the gray-scale facial template and the test image.

  • Step 2:

    Feature Points Extraction

    Taking face feature such as location of the eye, nose, mouth, including the face contour.

  • Step 3:

    Depth Estimation of Feature Points

    Estimation of the depth of a feature point with respect to other feature points.

  • Final Step:

    Facial Image Synthesis

    Generate 3D facial model based on the 2D fron-view image as a reference.

The metaverse can combine emerging technologies such as artificial intelligence (AI), augmented reality (AR), virtual reality (VR), and cloud and edge computing to create innovative solutions. Some of the ways are:

  • In 3D movies, special glasses are used to stimulate stereoscopic vision that add more depth to users’ brain. For example, when people use special glasses, they can see cars flying around or snowflakes floating in the air. 3D displays, on the other hand, are designed to simulate depth using stereoscopic vision, presenting two different images so that the viewers’ eyes interpret the images as a single 3D image. Window Shopping Implementation

Virtual Marketing

Window shopping will be the future of shopping, which makes consumers able to make purchases from a screen in front of each store for hours. Using 3D imaging technology, there will be a system that recognizes facial gestures and hand position, so shoppers can control a digital shop window display. The system supports transactions and can collect data on shopping trends without collecting personal information like facial recognition. The proposed interactive shop window technology differs from existing touchscreen shop window technology in that it employs a number of cameras to generate two stereo or 3D pictures, which are then processed by visualization software to operate the display. One stereo image depicts the face, while the other focuses on hand motions. The window shopper can zoom in on a product by staring at it and performing an inward sweeping motion, then using another gesture to see what it would look like in another color.
